# **Intelliarts Test Work**

## **Running Locally**

**Make sure you have Node.js installed.**

```
git clone https://github.com/ivan-hrynkiv/intelliarts-express # or clone
cd intelliarts-express
npm install
node src/app.js
```

---

## **How to use**

Start server:
**node app.js**

## Open application:

[Open in browser](http://localhost:3000/)
